Vue 提供了 minxin 来支持组件中的复用和组合功能。minxin 对象可以包含到任意组件选项。当组件使用混入对象时，所有混入对象的选项将被“混合”进该组件本身的选项。 当遇到同名选项时，这些选项也会以恰当的方式。

它们会遵循以下的合并方案

  • 数据对象:会在内部进行递归合并，并在发生冲突时以组件数据优先
  • 钩子函数：同名的钩子函数将合并为一个数组，它们都会被调用。且组件自身的钩子函数会最后调用
  • 值对象冲突：在值为对象的情况下冲突，例如 method、components 等，组件和混入对象的会被合并成一个对象，并在有冲突的情况时以组件对象的值为准。 
          
            var mixin = {
        created: function() {
          console.log("mixin function create lifecycle");
        },
        data: function() {
          return {
            language: "java、kotlin、javascript、go、sql",
            scope: "fron end、back end、media、framwork"
          };
        }
      };
      new Vue({
        mixins: [mixin],
        data: function() {
          return {
            skill: "english、search、doc、design ",
            language: "java"
          };
        },
        created: function() {
          console.log("this is a component created function");
        },
        methods: {
          funComponent: function() {
            console.log("function components");
          }
        }
      });
