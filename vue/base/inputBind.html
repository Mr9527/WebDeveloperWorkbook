修饰符

表单的输入绑定也如事件一样提供了大量的修饰符来让 model 能够更加专注的处理业务，常用的修饰符如下： 
        
          <!-- v-model 会在每次 input 事件后将输入框的值与数据进行同步。lazy 则可以让其转变为"change"时触发 -->
          <input v-model.lazy="message" type="text">               
          <!-- 即使 type='number' 的时候，元素也仍然会返回一个字符串，通过 number 则可以自动转换为数值 -->
          <input v-model.number="message" type="number">               
          <!-- trim 可以自自动过滤掉首位的空白字符 -->
          <input v-model.trim="message">
